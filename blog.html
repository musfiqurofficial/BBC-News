Var VS Let

NO VAR LET
01 The var is a keyword that is used to declare a variable The let is also a keyword that is used to declare a variable.
02 The variables that are defined with var statement have a function scope. The variables that are defined with let statement have a block scope.
03 var is an ECMAScript1 feature. let is a feature of ES6.
04 Syntax -: var name = value; Syntax -: let name = value;

Function VS Arrow Function

Function Arrow Function
Function is an ECMAScript1 feature. Arrow Function is a feature of ES6.
Syntax -: let x = function function_name(parameters){ }; Syntax -: let square = function(x){ return (x*x); };

Template literals (Template strings)

Template literals are literals delimited with backtick (`) characters, allowing for multi-line strings, for string interpolation with embedded expressions, and for special constructs called tagged templates.

Syntax:
`string text`
`string text line 1
string text line 2`
`string text ${expression} string text`
tagFunction`string text ${expression} string text`
